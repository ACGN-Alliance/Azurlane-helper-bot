> (快来点人帮咱开发吧呜呜呜，写代码好折磨)

本仓库使用的数据来自于[nonebot-plugin-azurlane-assistant-data](https://github.com/ACGN-Alliance/nonebot-plugin-azurlane-assistant-data)，为作者自爬数据  
## 功能
- :ship: 模拟建造(支持轻型重型特型以及限定池)
- :airplane: 装备参数查询

## 实用功能
- :black_large_square: 各功能的黑名单
- :1234: 自动化数据同步
- :smile: 基础关键词回复
- :dvd: Gitee 国内镜像源数据
